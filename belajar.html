CATATAN DALAM BELAJAR CODING

Kali ini saya akan mencatat beberapa tag tag yang ada dalam coding

contohnya:

1.Setiap awal ingin membuat sebuah codingan/sebuah program dibutuhkan tag.

tag nya yaitu !DOCYTYPE html

2.Didalam DOCYTYPE html tersebut terdapat beberapa unsur,seperti:

A.bagian kepala yaitu:head

B.bagian badan yaitu:dengan tag body

3.Tag tag yang terdapat pada sebuah web dan fungsinya:

A.Tag Title>

tag title berfungsi memberikan informasi berupa judul dokumen html

B.Tag P

Tag P tugasnya adalah membuat sebuah paragraf

C.Tag H1-H6

Fungsi tag H1-H6 adalah untuk memberikan judul sebuah paragraf

NOTE:semakin kecil angkanya maka akan semakin beesar pula judulnya

D.Tag a.

fungsi tag a/element a digunakan untuk mendefinisikan sebuah hyper link.

Contoh:
INI ADALAH CONTOH HYPERLINK.

E.Fungsi Tag IMG SRC ALT WIDTH and HEIGHT

Fungsi tag img src adalah untuk melampirkan sebuah gambar.

Contoh:

nah yang diatas ini menggunakan format img src
dan untuk lebar dan panjangnya menggunakan weight dan height
